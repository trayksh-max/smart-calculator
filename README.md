# 🚀 NovaCalc | The Next-Gen Smart Calculator

NovaCalc is not just a calculator—it's a high-energy, educational tool designed specifically for students. Moving beyond simple arithmetic, NovaCalc acts as a math tutor that helps students understand the *how* and *why* behind every answer.

## ✨ Features

### 🧠 Genius Mode (Algebraic Solver)
Unlike standard calculators, NovaCalc can handle symbolic math and multiple variables.
- **Variable Rules:** Define relationships like `y = 2x` or `z = x + 5`.
- **Substitution:** The calculator automatically substitutes these rules into complex equations.
- **Equation Solving:** Solve for `x` in equations like `5x + 2y = 18`.

### 💡 Educational Breakdown
Every result comes with a **Step-by-Step Breakdown** panel that explains the process:
1. **Substitution:** Shows exactly which variables were replaced.
2. **Simplification:** Shows the cleaned-up version of the expression.
3. **Solving:** Walks through the final steps to isolate the variable.

### 📜 Smart Tape (History)
Never lose a calculation again. The Smart Tape remembers your history, allowing you to click any previous result to bring it back to the display.

### 🎨 Neon-Bold Aesthetics
Designed for the modern student with a "Gamer/Neon" vibe:
- **Glassmorphism:** Frosted glass UI elements.
- **Glowing Accents:** Neon cyan, electric pink, and lime green highlights.
- **Interactive Animations:** Bouncy buttons and smooth transitions.

## 🛠️ How to Run
No installation required! NovaCalc is a web-based application.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/trayksh-max/smart-calculator.git
   ```
2. **Open the project:**
   Navigate to the folder and double-click `index.html` to open it in any modern web browser.

## ⌨️ Keyboard Shortcuts
For maximum speed, use your keyboard:
- **0-9 / .** : Input numbers
- **+, -, *, / , %** : Input operators
- **Enter / =** : Calculate result
- **Backspace** : Delete last character
- **Esc** : Clear all
